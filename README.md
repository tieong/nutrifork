# Sample Project - Test Branch

Un projet exemple simple créé pour démontrer un push sur la branche test.

## Description

Ce projet contient une page web simple avec:
- HTML5 sémantique
- CSS moderne avec animations
- JavaScript interactif
- Design responsive

## Structure des fichiers

```
/
├── index.html      # Page principale
├── styles.css      # Feuille de styles
├── script.js       # Logique JavaScript
└── README.md       # Ce fichier
```

## Fonctionnalités

- **Design Responsive**: S'adapte à toutes les tailles d'écran
- **Interactions**: Bouton cliquable avec messages aléatoires
- **Animations**: Effets visuels au chargement et au survol
- **Compteur**: Compte le nombre de clics avec des milestones

## Installation

1. Clonez le repository
2. Ouvrez `index.html` dans votre navigateur
